## Features
### Quran Reader
- Display Quran with Ottoman (Uthmani) script
- Multiple font options (Amiri, Noto Naskh, Uthman Taha)
- Font size controls
- Bismillah display at surah start

### Audio Player
- Multiple reciters support
- Auto-play next ayah
- Repeat functionality
- Playback speed control

### Prayer Times
- Automatic prayer times based on city
- Hijri date display
- Multiple Yemeni cities support

### UI/UX
- Professional HD design with emerald green and gold theme
- Dark/Light mode toggle
- Responsive layout for mobile and desktop
- Smooth animations
