# 📖 المصحف الإلكتروني - Quran Application

تطبيق ويب متكامل للقرآن الكريم مع التفسير والاستماع وأوقات الصلاة.

A comprehensive web application for the Holy Quran with Tafsir, Audio recitation, and Prayer times.

## ✨ المميزات - Features

### 📖 القرآن الكريم - Quran
- عرض القرآن الكريم بالخط العثماني
- التفسير (الزيادية وغيرها)
- البحث في الآيات
- تحليل الكلمات

### 🎧 الاستماع - Audio
- عدة قراء
- التكرار والتشغيل التلقائي
- التحكم في سرعة القراءة

### 🕌 أوقات الصلاة - Prayer Times
- أوقات الصلاة التلقائية حسب المدينة
- التاريخ الهجري

### 📜 الصحيفة السجادية - Sahifa Sajjadiya
- الأدعية المأثورة

## 🛠️ التقنيات المستخدمة - Tech Stack

- **Next.js 16** - React Framework
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **Prisma** - Database ORM
- **Zustand** - State Management
- **shadcn/ui** - UI Components

## 🚀 البدء - Getting Started

```bash
# تثبيت التبعيات
bun install

# تشغيل السيرفر
bun run dev

# بناء المشروع
bun run build
```
